# 2.2 for语句

## for语句

#### 1. 语法

for语句的语法形式是：

```c++ {.line-numbers}
for (初始化条件变量; 条件判断; 条件更改) {
    循环体语句
}
```

* **初始化条件变量**：初始化循环体条件变量，
* **条件判断**：循环控制条件，为真就会执行一次 循环体语句
* **条件更改**：通常是修改初始化的变量
  过程： 初始化条件变量 ->  条件判断 ->  true: 执行循环体  -> 条件更改 -> 回到 条件判断
                                   -> false: 结束for循环

#### 2. for语句头中的多重定义

**初始化条件变量**中可以定义多个变量，但是只能有一条声明语句，变量用逗号隔开，因此所有变量的基础类型必须相同：
**条件更改** 中可以修改多个变量，中间用逗号隔开（用逗号隔开的语句，不是多条语句哦，也是一条语句）。
```c++ {.line-numbers}
for (int i=0,j=0; i<10; i++,j+=2) {
    cout<<"i="<<i<<", j="<<j<<endl;
}
```

## for语句嵌套

for语句可以嵌套for语句
```c++ {.line-numbers}
for (int i=0; i<10; i++) {
    for(int j=0; i<i; j++) {
        cout<<"i="<<i<<", j="<<j<<endl;
    }
}
```
